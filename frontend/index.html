Guia de uso del aplicativo

Primer paso

Debes de encender mediante el interruptor nuestro prototipo para posteriormente ingresar situarlo sobre la planta.

Segundo paso

Debes consultar en la pagina web los datos obtenidos por el prototipo de tu planta pulsando el boton obtener datos.

Si se requiere puedes guardar los datos obtenidos en la base de datos pulsando el boton de Guardar datos

Tercer paso

Para poder calcular los litros de agua necesarios para regar tu planta debes pulsar el boton Calcular litros de agua

Cuarto paso

Si se requiere hemos integrado una IA para hacer recomendaciones personalizadas sobre los datos de tu planta o cualquier consulta relacionada con la misma. Las posibilidades son infinitas no las desaproveches.

Gracias

Muchas gracias por usar nuestro aplicativo, fue un placer ayudarle con su planta.
